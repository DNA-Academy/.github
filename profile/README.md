### Learning model

DNA Academy programs combine three main components:

1. **Theoretical sessions (online)**  
   - Core concepts in genomics, molecular biology and bioinformatics  
   - Case studies based on up-to-date literature  
   - Live lectures + recordings for later access  

2. **Genomic data analysis (online, hands-on)**  
   - Step-by-step workflows on Google Colab and similar tools  
   - NGS data formats, QC, alignment, variant analysis, microbiome/metagenomics, single-cell etc.  
   - Participants run all commands on their own computers  

3. **Wet-lab applications (onsite, in selected programs)**  
   - DNA/RNA isolation, PCR, library preparation  
   - 3rd generation sequencing (especially Oxford Nanopore)  
   - Real samples and field-driven experiments
